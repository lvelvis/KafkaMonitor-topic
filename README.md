# KafkaMonitor

#### 介绍
基于Python结合pykafka实现kafka生产及消费速率&主题分区偏移实时监控

实时采集Kafka生产者主题生产速率，主题消费速率，主题分区偏移，消费组消费速率，支持同时对多个来自不同集群的主题进行实时采集，支持同时对多个消费组实时采集
